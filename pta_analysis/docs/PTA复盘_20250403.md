**条件**：

**条件**：
- 价格先突破5300（可能是假突破）

- 价格回到5300下方并企稳 → 做空
- 止损：5310上方

### 方案C：企稳做多（低概率）

**条件**：
- 价格在5162-5300区间下沿（5162-5200）企稳
- 出现看涨吞没、锤子线等底部信号
- 5200附近企稳

**操作**：
- 5200附近轻仓做多，止损5150
- 目标5300

**注意**：此方案赔率较差，因为5300天花板明显
