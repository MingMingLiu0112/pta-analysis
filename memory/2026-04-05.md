## 深夜发现（17:41）

### 核心发现：czsc找最极端分型，不只是第一个
Czsc的笔终点选取逻辑：
- UP方向：找所有候选G，取**最低low**的那个
- DOWN方向：找所有候选D，取**最低high**的那个
- 这样就能把多个小笔合并成大笔

关键代码：
```python
# UP: 选最低low的G
fx_b = max(fxs_b, key=lambda fx: fx.high)
# DOWN: 选最低high的D  
fx_b = min(fxs_b, key=lambda fx: fx.low)
```
