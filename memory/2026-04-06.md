## 缠论核心规则（已确认）

### 线段破坏三规则
1. **笔破坏**：上升线段中，笔终点跌破上升线段最低点 → 线段结束
2. **反向走势破坏**：笔破坏后，反向走势不足3条线段 → 原线段延续
3. **破坏点范围**：回到破坏点起点范围内 = 破坏成立

### 中枢计算
- 中枢 = 连续3条线段的重叠区间
- ZG = min(各段GG), ZD = max(各段DD)
- 中枢区间 = [ZD, ZG]

## 线段破坏核心逻辑（用户讲解）

### 特征序列分型判断
- 底分型：三根相邻K线端点依次是高高低低高高（中间K线高低都是最低）
- 顶分型：三根相邻K线端点依次是低低高低高低（中间K线高低都是最高）

### 线段破坏两种情形
**常规情形（缺口封闭）**：
- 上行线段回调笔低点 < 前一上行笔高点 → 特征序列缺口封闭
- 反弹上行笔不新高再回落成下行笔，低点 < 前一下行笔低点（特征序列成顶分型）
- → 上行线段在顶分型高点确认结束

**非常规情形（缺口未封闭）**：
- 上行线段回调笔低点 > 前一上行笔高点 → 特征序列缺口未封闭
- 即使反弹回落形成顶分型，**也不能**确认上行线段结束
- 必须等到：该顶分型后的下行线段中，特征序列（上行笔）出现底分型
- → 才能同时确认：上行线段在顶分型结束，下行线段在底分型结束
- 这有点"未来函数"的意思（需要等后续走势确认）

### 关键词
- 缺口封闭 vs 未封闭 → 决定是否立即确认
- 特征序列成顶/底分型 → 确认线段终点的信号

## 下午补充2：线段确认的非常规情形

### 4月3日实例（线段1→线段2转折）
- bi4低点=6882 > bi2高点=6906 → 缺口未封闭（非常规）
- bi6出现：特征序列（上行笔bi5-bi7-bi9）形成底分型
- → 非常规确认：线段1在9:58结束，线段2在10:36结束，线段3开始

### 底层逻辑
- 缺口未封闭 = 空头力量弱，多头未真正反转
- 不能急于确认终点，要等空头衰竭（特征序列底分型）才能判断
- 这保证了结构的合理性，买卖点胜率更有保证

### 自动算法思路
1. 遍历所有转折候选点
2. 检查缺口封闭与否
3. 常规：立即确认
4. 非常规：等后续特征序列底/顶分型确认
5. 两步走：先标记候选，再验证
